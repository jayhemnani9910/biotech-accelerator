# Security Policy

## Supported versions

This project is pre-1.0. Security fixes are applied to `main` only.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, use GitHub's [private vulnerability reporting](https://github.com/jayhemnani9910/biotech-accelerator/security/advisories/new) to submit a report privately.

Please include:

- A description of the vulnerability and its impact
- Steps to reproduce
- Affected versions/commits
- Any suggested mitigation

You can expect an initial response within 7 days. We'll work with you to assess and address the issue, and credit you in the fix unless you prefer to remain anonymous.
